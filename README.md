# Patronero 

Patronero is open source software for donation mining using Monero cryptocurrency. Patronero is just end-user friendly [xmrig](https://github.com/xmrig/xmrig) wrapper with some additional functionality.


## OpenAlias

Patronero supports [OpenAlias](https://openalias.org/) so donators can add proteges by using their URL that has TXT record with OpenAlias specification.

## Run

Install PyQt5
``` pip3 install PyQt5 ```

Run project
```python3 application.py```

## Build

Install pyinstaller
``` pip3 install pyinstaller ```

Next build project to one file
``` pyinstaller application.py -n patronero --onefile ```
